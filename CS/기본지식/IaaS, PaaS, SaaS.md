- **IaaS(Infrastructure-as-a-Service)**
  - 인프라형 클라우드 서비스
  - 빈 방(인프라)을 제공해주고 필요한 것들을 개발자가 직접 설치
  - ex) aws의 ec2
- **PaaS(Platform-as-a-Service)**
  - 플랫폼형 클라우드 서비스
  - node.js, mongoDB 등이 설치되어있고 클릭을 통해 서비스 이용 가능
  - 모니터링, ci/cd 제공
  - ex)heroku
- **SaaS(Software as a Service)**

  - 서비스형 클라우드 서비스
  - 완전한 서비스를 제공받아 사용
  - ex) google docs

- **IaaS vs PaaS**
  - 유연성 - IaaS > PaaS
  - 이식성 - IaaS > PaaS
  - 운영비효율 - IaaS < PaaS
